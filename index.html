How it works

JumpCutter detects frames with silence and automatically removes them. It also
detects when you're clapping and takes a frame at thet moment.


If you want to use it by yourself, complete the following steps:

  • 1. Download and install Python 3.7
  • 2. Download JumpCutter.
  • 3. Open "install-requirements.bat".

    • You're done!


    How to use

  • 1. Open CMD by typing "cmd" in the search bar.
  • 2. Type "cd (JumpCutter directory)".
  • 3. Type "jumpcutter.py --input_file (filename) --output_file (filename).

    • Additional options:

    If you want to edit a video with a different frame rate (e.g. 24 or 60 fps), then
    type "--frame_rate (frame rate)" to the end of the line.

    If you want to make silent fragments to play at a different speed (e.g. 3x or 10x), then
    type "--silent_speed (silent speed with .00 at the end, e.g. 5.00)" to the end of the line.

    If you want to edit a video with a different sample rate (e.g. 36000 Hz), then
    type "--sample_rate (sample rate)" to the end of the line.

    If you want to set the volume amount that frames' audio needs to surpass to be consider
    "sounded" (it ranges from 0 (silence) to 1 (max volume)), then type "--silent_threshold (silent threshold)" to the end of the line.

    If you want to make sounded fragments to play at a different speed (e.g. 2x or 3x), then
    type "--sounded_speed (sounded speed with .00 at the end, e.g. 2.00)" to the end of the line.

    If you want to edit a video with a different frame margin (e.g. 2 or 3), then
    type "--frame_margin (frame margin)" to the end of the line.

    If you want to set the quality of frames to be extracted from input video (1 is highest, 31 is lowest), then
    type "--frame_quality (frame quality)" to the end of the line.
